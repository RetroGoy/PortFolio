# Portfolio

Ce portfolio est une application React conçue comme une interface
inspirée d’un système d’exploitation à l’ancienne.

Le projet m’a permis d’expérimenter des interactions autour de fenêtres
déplaçables et organisables,
tout en construisant une base claire et scalable pour présenter
différents projets au même endroit.

## Fonctionnalités

- Interface de type “OS”
- Fenêtres déplaçables et redimensionnables
- Organisation modulaire des projets
- Animations et interactions UI

## Stack technique

- JavaScript
- React
- Tailwind CSS

## À propos du projet

Ce portfolio a servi de terrain d’apprentissage pour mieux comprendre
la structuration d’une application React,
la gestion de composants réutilisables
et la mise en place d’une base évolutive
permettant d’ajouter facilement de nouveaux projets.
